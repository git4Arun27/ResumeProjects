Coda
- Developed an end-to-end object detection system using YOLOv8 for a machine learning product.  
- Created a Flask-based backend to interact with YOLOv8 model weight files and manage object detection workflows.  
- Designed and implemented file upload and download APIs using Java Spring Boot as a core component of the project.  
- Containerized the entire application using Docker, creating images for deployment.  
- Deployed the containerized application, including the file upload and download APIs, on Render for production hosting.  

**Tech Stack:** Flask, Spring Boot, YOLO, MySQL, MongoDB, Docker.
